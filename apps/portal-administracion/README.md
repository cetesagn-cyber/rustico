# Portal de Administracion

Portal interno para administracion y recepcion de Rustico.

## Incluye

- Dashboard operativo.
- Agenda y estados de citas.
- Clientes e historial.
- Catalogo de servicios, productos y combos.
- Barberos, usuarios y permisos.
- Financiero, adelantos, estadisticas y WhatsApp.

## Desarrollo

```bash
npm install
npm run dev
```

URL local: http://localhost:5180

## Build

```bash
npm run build
```
