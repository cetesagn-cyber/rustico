# App Barberos

App movil/PWA para el trabajo diario de los barberos.

## Incluye

- Login de barbero.
- Agenda del dia.
- Clientes asociados.
- Perfil del barbero.
- Navegacion optimizada para celular.

## Desarrollo

```bash
npm install
npm run dev
```

URL local: http://localhost:5181

## Build

```bash
npm run build
```
